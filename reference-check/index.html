Reference check fraud detection

Backy Check's algorithm monitors multiple data touch-points throughout the referencing process and will notify you of any unusual activity. As it does so, if anything suspicious is detected, the system will raise a flag that some unusual activity has occurred. Reasons for a flag include matching IP addresses for the candidate and their referee, matching operating systems or identical internet browsers, among others.
